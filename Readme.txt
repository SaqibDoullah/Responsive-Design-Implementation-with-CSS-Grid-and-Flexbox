# QR Code Component Project

This project is a simple QR Code component developed for the Frontend Mentor challenge. It showcases a static QR code that links to the Frontend Mentor website to help improve front-end development skills.

## Overview

The QR Code Component displays a QR code which users can scan to be redirected to a specific webpage. This project aims to provide a clean, responsive design that adapts to different device screens.

## Features

- **Responsive Design**: The QR code component is fully responsive and adjusts to fit mobile, tablet, and desktop views.
- **Accessibility Features**: Basic accessibility features include clear contrast and keyboard navigability.

## Technologies Used

- HTML5
- CSS3
